About Course

Front end web developement is everything involved in programming the user interface of web application. Typically it refers to Hypertext Markup Language(HTML),Cascading Style Sheets(CSS) and JavaScript portion of web site production as opposed to database or server-side programming. It encompasses everything from building simple page of HTML text to creating complex responsive HTML5 websites designed to be accessed via various browsers,devices and screen sizes.
